Le framework

Next.js

Next.js est un framework full-stack basé sur React.

Voici quelques-unes de ses spécificités :

  • Système de route basé sur l'arborescence des fichiers
  • Création de composants côté serveur ou côté client (application hybride)
  • Optimisation des performances (mise en cache des données, optimisation des images, des fonts...)
  • Support de TypeScript (optionnel)
  • Communauté importante
  • Mis en avant dans la doc de React et collaboration des dev React/Next.js

Vercel

Next.js est développé par Vercel, une entreprise qui propose un service de déploiement de sites web.

Leur service est spécialisé dans le déploiement de sites web basés sur Next.js, mais il est également possible de déployer des sites web basés sur d'autres technologies (NodeJS, Python, Ruby, Go...).

Le déploiement est simplifié grace au CLI que propose l'entreprise.

Versions

La version actuelle de Next.js est la version 14.

Au cours des derniers mois, Next.js a beaucoup évolué sur la gestion des server components ainsi que sur les performances dans l'exécution (système de cache) mais également sur la compilation avec des outils basés sur le langage Rust (SWC et TurboPack).

Alternatives

Le principal concurrent de Next.js est Remix, un autre framework basé sur React et très similaire à Next.js.

Il existe également des alternatives plus légères comme GatsbyJS, qui est un générateur de site statique basé sur React.
